Post Collapse Systems - bootstrap template

This directory contains the minimal files needed to bootstrap a Jekyll-style book site similar to this repository.

Files:

- `_config.yml` - site configuration
- `index.md` - home page
- `_layouts/default.html` - minimal layout
- `_includes/head-custom.html` - minimal head includes
- `book/manifest.txt` - example book manifest

How to use:

1. Copy the `bootstrap-template` folder into a new project.
2. Install Jekyll (if needed) and build the site.
3. Add your content under `book/` and expand `_includes` and `_layouts` as required.
